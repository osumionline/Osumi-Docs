Empezar

Osumi Framework proporciona un conjunto de herramientas y una forma de diseñar aplicaciones que permite el desarrollo rápido y modular de aplicaciones web tanto pequeñas como grandes.

Al estar diseñado de una manera orientada a "componentes" o pequeñas piezas reutilizables, se busca la eficiencia y que la ejecución de una aplicación solo utilice aquellos recursos y conceptos estríctamente necesarios.

Por ejemplo, una aplicación que contenga mas de mil componentes, solo cargará y usará aquellos necesarios para la ejecución solicitada, mientras que el resto de componentes no será necesario cargarlos resultando en un impacto de recursos mínimo.

Esto se consigue con el uso de Composer y su sistema de "auto-load" que carga los recursos a medida que se van necesitando.

Software libre

Osumi Framework se distribuye como software libre de modo que es abierto y gratuito para todo el mundo. Está sujeto a la licencia MIT de modo que se permite su uso personal y comercial, ser modificado y distribuido, limitando la responsabilidad y la garantía a sus usuarios.

El código del framework se encuentra disponible en Github, donde se organiza mediante releases en las que se agrupan las novedades, cambios o correcciones disponibles.

Requisitos

Para poder usar Osumi Framework es necesario cumplir una serie de requisitos técnicos:

  • PHP: versión 8.3 o superior
  • Composer: versión 2.8 o superior

Osumi Framework ha sido creado y diseñado para ser usado con Apache 2.4 o superior, pero en principio no habría ningún impedimento para que sea usado con cualquier otro servidor que soporte la ejecución de código PHP (tal vez requiera adaptar el archivo .htaccess que hace de controlador frontal).

Bases de datos: para acceder a las bases de datos, Osumi Framework utiliza las librerías nativas PDO proporcionadas por PHP, de modo que es transparente el uso de cualquier motor de base de datos. Ha sido diseñado y probado para ser usado con MySQL/MariaDB 10.5 o superiores, pero en principio podría ser usado con PostgreSQL, SQLite...

Instalación

Para crear una nueva aplicación usando Osumi Framework, simplemente hay que ejecutar el siguiente comando:

composer create-project osumionline/new

Tras ejecutar este comando, se habrá creado una carpeta llamada "new" que contendrá una aplicación de demostración y el código propio de Osumi Framework.

Esta aplicación de demostración sirve para orientar en la estructuración del código o como ejemplos de los distintos elementos que se pueden usar.

En la raíz de la aplicación también se habrá creado un archivo llamado "of", es el ejecutable (o CLI) que permite ejecutar tareas desde línea de comandos.

Entre las opciones de este CLI, hay un comando llamado "reset" que permite limpiar todos los datos de demostración y dejar todas las carpetas en un estado inicial:

php of reset
